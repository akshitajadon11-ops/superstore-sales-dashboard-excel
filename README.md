# 📊 Superstore Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project analyzes retail sales data using Microsoft Excel. An interactive dashboard was created to help understand sales performance, profit trends, customer segments, product categories, shipping modes, and regional performance.

## 🛠 Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* KPI Cards
* Helper Columns
* Conditional Formatting

## 📈 Dashboard Features

* Total Sales KPI
* Total Profit KPI
* Total Orders KPI
* Monthly Sales Trend
* Sales by Category
* Sales by Region
* Profit by Category
* Customer Analysis
* Shipping Mode Analysis
* Interactive Slicers

## 💡 Business Insights

* West Region generated the highest sales.
* Technology category produced the highest revenue.
* Consumer segment contributed the largest share of sales.
* Standard Class was the most frequently used shipping mode.
* Sales showed strong growth toward the end of the year.
